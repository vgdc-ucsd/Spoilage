# FMOD

## Audio Team

Make sure you're using FMOD Studio 2.03.xx, otherwise things will break

All you need to do is place built `.bank` files into `Assets/Audio/Banks` so programmers can play the contained events

## Programming Team

Unity's built-in audio system is disabled, we will be using FMOD

If you wish to play a sound effect, use the `FMOD Studio Event Emitter` component
