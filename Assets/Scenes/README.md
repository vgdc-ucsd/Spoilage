# Scenes

## Persistent

IMPORTANT: Always have Persistent loaded in-editor alongside the scenes you're working on. You can load multiple scenes by either:
- Right-click scene in Project Panel -> Click "Open Scene Additive"
- Click-and-hold scene in Project Panel -> Drag to Hierarchy Panel

Persistent should be the first scene loaded, and should always remain loaded.

It contains the Main Camera, EventSystem, and singletons that should remain accessible throughout the game.
